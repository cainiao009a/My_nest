# 基础nest开发框架配置

## Description
集成了Fastify、 版本控制（但请求控制以及全局请求控制配置）、全局返回参数、全局异常拦截、环境配置、热重载、文档生成，等基础脚手架配置。
文档内容通过('localhost:XXXX/api/doc')访问

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development hot start
$ npm run start:hotdev

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
